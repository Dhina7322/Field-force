Design a modern SaaS web application UI for a "Field Contractor Workforce Management System".

This system is used by contractors managing grass cutting and field work operations.

---

LAYOUT:

* Desktop (1440px)
* Left dark sidebar
* Top header
* Main content area

---

SIDEBAR:

* Dashboard

SECTION: WORKFORCE

* Employees
* Teams

SECTION: WORK MANAGEMENT (highlighted)

* Work Assignments
* Daily Work Log

SECTION: ATTENDANCE

* Check In / Check Out
* Time Tracking

SECTION: MACHINES

* Machines
* Machine Usage

SECTION: CLIENTS

* Clients
* Work Areas

SECTION: FINANCE

* Salary Calculation
* Ledger

SECTION: SETTINGS

* Company
* Roles

---

MAIN SCREEN: WORK ASSIGNMENT PAGE

Header:

* Title: Work Assignments
* Button: + Create Work

Create Work Form:

Fields:

* Client (dropdown)
* Area (dropdown)
* Date
* Start Time
* End Time
* Team (dropdown)
* Employees (multi-select)
* Machine (dropdown)
* Notes

---

WORK LIST VIEW:

Cards or Table showing:

* Client Name
* Area
* Assigned Team
* Employees count
* Machine used
* Status (Planned / Ongoing / Completed)
* Total Hours

---

ATTENDANCE UI:

* Check-in button
* Check-out button
* Break start / end
* Live working timer

---

DESIGN STYLE:

* Clean, modern SaaS UI
* Dark sidebar (#0F172A)
* Light content (#F8FAFC)
* Green/blue accents for status
* Rounded cards (8px)
* Soft shadows
* Font: Inter

---

EXTRA:

* Focus on real-world usability
* Large buttons for field usage
* Simple and fast interactions
* Mobile-friendly adaptability
